# Task Files

The demo task expects each task to be represented by a directory at:

```text
tasks/<task-id>/
```

Each task directory should contain:

- `task.json`: machine-readable task metadata
- `prompt.md`: the prompt or instruction shown to the agent

## Expected Branch And Title Shape

For task `demo-001`, the expected pull request shape is:

- branch: `task/demo-001`
- title contains: `[task:demo-001]`

## Example Task JSON

```json
{
  "task_id": "example-001",
  "summary": "add a helper and tests",
  "prompt": "Open a PR that adds a small helper function and tests.",
  "base_branch": "main",
  "head_branch": "task/example-001",
  "pr_title_token": "[task:example-001]",
  "required_checks": ["lint", "test"],
  "deadline": "2026-04-11T18:00:00Z",
  "payout_sats": 100000
}
```

The exact task file contents can be generated by the Five Bells demo
coordinator. The example directory in this repository is only a seed shape.
